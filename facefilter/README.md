# Installation
Copy happy.png + haarcascade + py file somewhere (like /opt/face)

Install with `sudo ln -s <absolute-path-of-face.py> /usr/bin/face` 

# Usage
`face a.jpg b.jpg` -> will create `_a.jpg`, `_b.jpg`

# Requirements
* opencv2
* cv2-python bindings
* PIL or Pillow
